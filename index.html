محمد ارمین - توسعه‌دهنده وب، بازی‌ساز و برنامه‌نویس

درباره من

سلام! من محمد ارمین هستم، توسعه‌دهنده وب، بازی‌ساز و برنامه‌نویس با چند سال تجربه در زمینه‌های مختلف فناوری اطلاعات.

علاقه‌مند به خلق پروژه‌های نوآورانه و کاربرپسند هستم که بتوانند ارزش واقعی برای کاربران ایجاد کنند.

هدف من ارتقاء مهارت‌هایم و ارائه بهترین راه‌حل‌های نرم‌افزاری به مشتریان و تیم‌های توسعه است.

مهارت‌ها

  • تسلط به HTML، CSS و JavaScript برای طراحی صفحات وب مدرن و واکنش‌گرا.
  • تجربه کار با React و Vue برای توسعه برنامه‌های پیشرفته تحت وب.
  • توسعه Backend با Node.js و Express و مدیریت پایگاه داده‌ها.
  • طراحی بازی با Unity و Unreal Engine برای ساخت بازی‌های سه‌بعدی و دو‌بعدی.
  • برنامه‌نویسی با زبان‌های C# و Python برای پروژه‌های متنوع.

نمونه کارها

وبسایت فروشگاهی مدرن

طراحی یک فروشگاه اینترنتی با رابط کاربری زیبا و کاربرپسند.

بهینه شده برای موبایل و دسکتاپ با بارگذاری سریع.

استفاده از تکنولوژی‌های به‌روز برای تجربه خرید راحت.

بازی پلتفرمر دوبعدی

ساخت بازی دوبعدی با گرافیک فانتزی و گیم‌پلی جذاب.

طراحی مراحل چالش‌برانگیز و سرگرم‌کننده برای کاربران مختلف.

پیاده‌سازی جلوه‌های بصری زیبا و انیمیشن‌های روان.

برنامه مدیریت پروژه تحت وب

ابزاری برای مدیریت وظایف و همکاری تیمی به صورت آنلاین.

نمایش داشبورد و نمودارهای پیشرفت پروژه‌ها.

طراحی رابط کاربری ساده و کارآمد برای کاربران مختلف.

تجربه واقعیت مجازی

ساخت محیط‌های سه‌بعدی تعاملی با کنترل‌های ساده.

امکان تعامل با اشیاء و انجام مأموریت‌ها در فضای VR.

بهینه‌سازی برای دستگاه‌های مختلف واقعیت مجازی.
